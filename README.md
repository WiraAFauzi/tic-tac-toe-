# Tic Tac Toe (Python)

A text-based Tic Tac Toe game playable in the command line.

## Features
- Player vs Player or Player vs AI
- AI difficulty levels: Easy, Medium, Hard
- Best-of series mode (first to 3, 5, or 10 wins)
- Score tracker
- Shows available moves each turn
- Exit anytime with `q`, `quit`, or `exit`

## How to Run
```bash
python tic-tac-toe.py
